# Wall Calendar Component

A React + Vite single-page component that recreates the wall-calendar aesthetic with a hero image, range selection, and integrated notes.

## Features
- Hero image + month label inspired by a physical wall calendar
- Click-and-drag style range selection (start + end with hover preview)
- Monthly notes and range-tied notes stored in localStorage
- Responsive layout for desktop and mobile
- Optional "flip" animation and theme toggle

## Run locally

```bash
npm install
npm run dev
```
